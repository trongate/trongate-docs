[meta-description]Trongate requires PHP 8.0 or higher, MySQL or MariaDB, Apache or Nginx with URL rewriting. No Composer, no Packagist, no external dependencies.[/meta-description]

Minimum System Requirements

Trongate is built for modern PHP. To run the framework, you will need:

Requirement Minimum Recommended
PHP 8.0 8.2 or higher
Database MySQL 5.7 / MariaDB 10.3 MySQL 8.0 / MariaDB 10.6
Web Server Apache 2.4 or Nginx 1.18 Apache 2.4+ / Nginx 1.24+
URL Rewriting mod_rewrite (Apache) or equivalent Enabled

Trongate is a Native PHP framework. There are no third-party dependencies, no Composer packages, and no Packagist repositories. Everything you need ships with the framework itself.
