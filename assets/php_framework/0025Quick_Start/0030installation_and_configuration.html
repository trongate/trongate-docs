Installation & Configuration

After downloading Trongate, installation takes only a few minutes. There are no dependency managers, no build steps, and no scaffolding commands. The framework includes an intelligent Setup Wizard that guides you through the entire process.

1. Place the Files on Your Server

Upload the contents of the Trongate ZIP file to your web server's document root. On many shared hosts this is a directory called public_html or www. The resulting structure should look like this:

[code] your-site.com/ ↵ config/ ↵ engine/ ↵ modules/ ↵ public/ ↵ uploads/ ↵ .htaccess ↵ index.php [/code]

The public/ directory is the web root. Point your domain to public/ if possible — this keeps the framework files one level above public access for better security.

From here, the Setup Wizard handles the rest. No manual file editing is required to get started.

2. Visit Your Site — The Setup Wizard Loads

Navigate to your site's URL. The Setup Wizard will automatically detect that Trongate has not yet been configured and will present a 5-step installation wizard:

Step 1 — Welcome

The welcome screen confirms your server meets the minimum requirements and presents an overview of the installation process. Click "Let's Go!" to proceed.

Step 2 — Database

Enter your database connection details — host, port, username, password, and database name. Click "Test Database Connection" to verify the settings are correct.

Tip: If your database does not yet exist, Trongate will attempt to create it automatically. Alternatively, create the database through your hosting control panel first and then enter its name here.

Step 3 — Confirm

After a successful connection test, you will see the exact PHP configuration that will be written to your config/database.php file. You can either:

  • Copy the code manually and save it to config/database.php, or
  • Let the wizard write it for you — it will save the file automatically

The wizard displays the code it will write so you retain full visibility and control.

Step 4 — Admin Account

Create your administrator account by providing a username and password. This account gives you full access to the Trongate admin panel. The wizard:

  • Creates the necessary database tables
  • Seeds the database with initial data
  • Registers your administrator account

Step 5 — Complete

Installation is finished. A confirmation message confirms everything is set up and the wizard module removes itself from the framework, leaving a clean installation. Click "Go to Admin Panel" to log in and start building.

What the Setup Wizard Does For You

  • Verifies PHP version and server requirements
  • Tests and configures your database connection
  • Writes the config/database.php file with your settings
  • Creates all required database tables
  • Seeds default data (including user levels and token system)
  • Creates your administrator account
  • Configures the BASE_URL in config/config.php
  • Removes itself after completion, leaving no trace

Manual Configuration (Optional)

If you prefer to configure Trongate manually, you can do so by editing the configuration files directly:

Configure Your Base URL

Open config/config.php and set the BASE_URL constant to the URL where your application will be accessed:

[code=php] <?php define('BASE_URL', 'http://localhost/trongate/'); define('ENV', 'dev'); [/code]

Warning: Never launch a Trongate application into a live or production environment with the ENV constant set to 'dev' or 'DEV'.

When ENV is set to 'dev', the framework may display detailed error messages and debugging information that could expose sensitive details about your application. For production, always set ENV to 'prod', 'live', or any value other than 'dev'.

Configure Your Database (if not using the wizard)

Open config/database.php and enter your database connection details:

[code=php] <?php $databases = [ 'default' => [ 'host' => '127.0.0.1', 'port' => '3306', 'user' => 'root', 'password' => '', 'database' => 'trongate' ] ]; [/code]

You can define multiple database groups by adding more entries to the $databases array. Additional guidance is available on our documentation on Working With Multiple Databases.

3. Verify the Installation

Navigate to the URL you set in BASE_URL. If you used the Setup Wizard, you will be redirected to the login page. If you configured manually, you should see the default Trongate welcome page.

That’s it. You’re ready to build.
