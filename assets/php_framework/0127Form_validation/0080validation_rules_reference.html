Validation Rules Reference

This page provides a complete reference of all built-in validation rules available in Trongate v2. Each rule validates a specific aspect of form data, and most rules are optional - fields only validate when they contain data (except required).

Optional Fields: All validation rules except required skip validation if the field is empty. To validate an empty field, use the required rule first.

Standard Rules

required

The field cannot be empty. This is the only rule that validates empty fields - all other rules skip validation if the field is empty.

Syntax:

[code=php] $this->validation->set_rules('email', 'email address', 'required'); [/code]

Error Message:

[code=text] The [label] field is required. [/code]

numeric

The field must contain a numeric value (integer or decimal, positive or negative).

Syntax:

[code=php] $this->validation->set_rules('age', 'age', 'numeric'); [/code]

Valid Values: 42, -15, 3.14, -2.5

Invalid Values: abc, 12x, 1,000

Error Message:

[code=text] The [label] field must be numeric. [/code]

integer

The field must contain a whole number (no decimal places, positive or negative).

Syntax:

[code=php] $this->validation->set_rules('quantity', 'quantity', 'integer'); [/code]

Valid Values: 42, -15, 0

Invalid Values: 3.14, abc, 42.0

Error Message:

[code=text] The [label] field must be an integer. [/code]

decimal

The field must contain a decimal number (with or without decimal places).

Syntax:

[code=php] $this->validation->set_rules('price', 'price', 'decimal'); [/code]

Valid Values: 19.99, 10, -3.5

Invalid Values: $19.99, abc

Error Message:

[code=text] The [label] field must be a decimal number. [/code]

Format Rules

valid_email

The field must contain a valid email address.

Syntax:

[code=php] $this->validation->set_rules('email', 'email address', 'valid_email'); [/code]

Valid Values: user@example.com, john.doe+tag@company.co.uk

Invalid Values: user@, @example.com, user name@example.com

Error Message:

[code=text] The [label] field must contain a valid email address. [/code]

valid_url

The field must contain a valid URL (with protocol).

Syntax:

[code=php] $this->validation->set_rules('website', 'website', 'valid_url'); [/code]

Valid Values: https://example.com, http://example.com/path

Invalid Values: example.com, not a url

Error Message:

[code=text] The [label] field must contain a valid URL. [/code]

valid_ip

The field must contain a valid IP address (IPv4 or IPv6).

Syntax:

[code=php] $this->validation->set_rules('ip_address', 'IP address', 'valid_ip'); [/code]

Valid Values: 192.168.1.1, 2001:0db8:85a3::8a2e:0370:7334

Invalid Values: 256.256.256.256, not an ip

Error Message:

[code=text] The [label] field must contain a valid IP address. [/code]

Date and Time Rules

valid_date

The field must be a valid date in YYYY-MM-DD format.

Syntax:

[code=php] $this->validation->set_rules('birth_date', 'birth date', 'valid_date'); [/code]

Valid Values: 2024-03-18, 1999-12-25

Invalid Values: 03/18/2024, 2024-13-01, 2024-02-30

Error Message:

[code=text] The [label] field must be a valid date in YYYY-MM-DD format. [/code]

valid_time

The field must be a valid time in HH:MM or HH:MM:SS format (24-hour).

Syntax:

[code=php] $this->validation->set_rules('meeting_time', 'meeting time', 'valid_time'); [/code]

Valid Values: 14:30, 09:15:45, 23:59:59

Invalid Values: 2:30 PM, 25:00, 14:60:00

Error Message:

[code=text] The [label] field must be a valid time in HH:MM or HH:MM:SS format. [/code]

valid_datetime_local

The field must be a valid datetime in YYYY-MM-DDTHH:MM format (ISO 8601 local datetime).

Syntax:

[code=php] $this->validation->set_rules('appointment', 'appointment', 'valid_datetime_local'); [/code]

Valid Values: 2024-03-18T14:30, 2024-12-25T23:59

Invalid Values: 2024-03-18 14:30, 03/18/2024T14:30

Error Message:

[code=text] The [label] field must be a valid datetime in YYYY-MM-DDTHH:MM format. [/code]

valid_month

The field must be a valid month in YYYY-MM format.

Syntax:

[code=php] $this->validation->set_rules('billing_month', 'billing month', 'valid_month'); [/code]

Valid Values: 2024-03, 2024-12

Invalid Values: 2024-13, 03-2024, March 2024

Error Message:

[code=text] The [label] field must be a valid month in YYYY-MM format. [/code]

valid_week

The field must be a valid week in YYYY-W## format (ISO 8601 week date).

Syntax:

[code=php] $this->validation->set_rules('work_week', 'work week', 'valid_week'); [/code]

Valid Values: 2024-W12, 2024-W52

Invalid Values: 2024-W53 (if year doesn't have 53 weeks), W12-2024

Error Message:

[code=text] The [label] field must be a valid week in YYYY-W## format. [/code]

Length Rules

min_length

The field must be at least the specified number of characters long.

Syntax:

[code=php] $this->validation->set_rules('password', 'password', 'min_length[8]'); $this->validation->set_rules('username', 'username', 'min_length[3]'); [/code]

Valid for min_length[3]: abc, john, user123

Invalid for min_length[3]: ab, x

Error Message:

[code=text] The [label] field must be at least [param] characters long. [/code]

max_length

The field cannot exceed the specified number of characters.

Syntax:

[code=php] $this->validation->set_rules('bio', 'biography', 'max_length[500]'); $this->validation->set_rules('city', 'city', 'max_length[50]'); [/code]

Valid for max_length[50]: New York, Los Angeles

Invalid for max_length[50]: Any string longer than 50 characters

Error Message:

[code=text] The [label] field cannot exceed [param] characters. [/code]

exact_length

The field must be exactly the specified number of characters long.

Syntax:

[code=php] $this->validation->set_rules('postal_code', 'postal code', 'exact_length[5]'); $this->validation->set_rules('zip', 'ZIP code', 'exact_length[5]'); [/code]

Valid for exact_length[5]: 12345, abcde

Invalid for exact_length[5]: 1234, 123456

Error Message:

[code=text] The [label] field must be exactly [param] characters long. [/code]

Comparison Rules

greater_than

The field must contain a value greater than the specified number.

Syntax:

[code=php] $this->validation->set_rules('age', 'age', 'greater_than[18]'); $this->validation->set_rules('quantity', 'quantity', 'greater_than[0]'); [/code]

Valid for greater_than[18]: 19, 25, 100

Invalid for greater_than[18]: 18, 17, 0

Error Message:

[code=text] The [label] field must be greater than [param]. [/code]

less_than

The field must contain a value less than the specified number.

Syntax:

[code=php] $this->validation->set_rules('discount', 'discount', 'less_than[100]'); $this->validation->set_rules('rating', 'rating', 'less_than[5]'); [/code]

Valid for less_than[100]: 99, 50, 0

Invalid for less_than[100]: 100, 101

Error Message:

[code=text] The [label] field must be less than [param]. [/code]

matches

The field must match the value of another field. Commonly used for password confirmation.

Syntax:

[code=php] $this->validation->set_rules('password', 'password', 'required|min_length[8]'); $this->validation->set_rules('confirm_password', 'password confirmation', 'required|matches[password]'); [/code]

Valid: Both fields contain the same value

Invalid: Fields contain different values

Error Message:

[code=text] The [label] field must match the [param] field. [/code]

Note: The parameter is the name of the field to compare against, not a label.

File Validation Rules

allowed_types

The uploaded file must be one of the specified file types (by extension).

Syntax:

[code=php] $this->validation->set_rules('document', 'document', 'allowed_types[pdf,doc,docx]'); $this->validation->set_rules('resume', 'resume', 'allowed_types[pdf,txt]'); [/code]

Parameters: Comma-separated list of allowed file extensions (without dots)

Valid for allowed_types[pdf,doc,docx]: resume.pdf, letter.doc, report.docx

Invalid for allowed_types[pdf,doc,docx]: image.jpg, video.mp4

Error Message:

[code=text] The [label] must be one of the following file types: [param]. [/code]

max_size

The uploaded file size cannot exceed the specified limit in kilobytes.

Syntax:

[code=php] $this->validation->set_rules('photo', 'photo', 'max_size[5000]'); // 5MB $this->validation->set_rules('document', 'document', 'max_size[10000]'); // 10MB [/code]

Parameters: Maximum file size in kilobytes

Error Message:

[code=text] The [label] file size must not exceed [param] KB. [/code]

min_size

The uploaded file size must be at least the specified limit in kilobytes.

Syntax:

[code=php] $this->validation->set_rules('video', 'video', 'min_size[100]'); // At least 100KB [/code]

Parameters: Minimum file size in kilobytes

Error Message:

[code=text] The [label] file size must be at least [param] KB. [/code]

Image Validation Rules

is_image

The uploaded file must be a valid image file (JPG, PNG, GIF, or WEBP).

Syntax:

[code=php] $this->validation->set_rules('avatar', 'avatar', 'is_image'); [/code]

Valid File Types: JPG, PNG, GIF, WEBP

Error Message:

[code=text] The [label] must be a valid image file. [/code]

max_width

The image width cannot exceed the specified number of pixels.

Syntax:

[code=php] $this->validation->set_rules('banner', 'banner', 'is_image|max_width[1920]'); [/code]

Parameters: Maximum width in pixels

Error Message:

[code=text] The [label] image width cannot exceed [param] pixels. [/code]

min_width

The image width must be at least the specified number of pixels.

Syntax:

[code=php] $this->validation->set_rules('thumbnail', 'thumbnail', 'is_image|min_width[200]'); [/code]

Parameters: Minimum width in pixels

Error Message:

[code=text] The [label] image width must be at least [param] pixels. [/code]

max_height

The image height cannot exceed the specified number of pixels.

Syntax:

[code=php] $this->validation->set_rules('banner', 'banner', 'is_image|max_height[600]'); [/code]

Parameters: Maximum height in pixels

Error Message:

[code=text] The [label] image height cannot exceed [param] pixels. [/code]

min_height

The image height must be at least the specified number of pixels.

Syntax:

[code=php] $this->validation->set_rules('thumbnail', 'thumbnail', 'is_image|min_height[150]'); [/code]

Parameters: Minimum height in pixels

Error Message:

[code=text] The [label] image height must be at least [param] pixels. [/code]

exact_width

The image width must be exactly the specified number of pixels.

Syntax:

[code=php] $this->validation->set_rules('logo', 'logo', 'is_image|exact_width[200]'); [/code]

Parameters: Exact width in pixels

Error Message:

[code=text] The [label] image width must be exactly [param] pixels. [/code]

exact_height

The image height must be exactly the specified number of pixels.

Syntax:

[code=php] $this->validation->set_rules('logo', 'logo', 'is_image|exact_height[200]'); [/code]

Parameters: Exact height in pixels

Error Message:

[code=text] The [label] image height must be exactly [param] pixels. [/code]

square

The image must be square (equal width and height).

Syntax:

[code=php] $this->validation->set_rules('profile_pic', 'profile picture', 'is_image|square'); [/code]

Valid: Image with width = height (e.g., 200x200, 500x500)

Invalid: Any rectangular image (e.g., 400x300, 1920x1080)

Error Message:

[code=text] The [label] image must be square (equal width and height). [/code]

Combining Rules

Multiple rules can be combined using the pipe character (|):

[code=php] // Email must be required, valid email format $this->validation->set_rules('email', 'email address', 'required|valid_email'); // Username must be required, 3-20 characters $this->validation->set_rules('username', 'username', 'required|min_length[3]|max_length[20]'); // Image must be valid image, square, max 2MB $this->validation->set_rules('avatar', 'avatar', 'is_image|square|max_size[2000]'); // Password must match confirmation $this->validation->set_rules('password', 'password', 'required|min_length[8]'); $this->validation->set_rules('confirm_password', 'password confirmation', 'required|matches[password]'); [/code]

Early Exit: Validation stops at the first error for each field. If required fails, other rules for that field are not checked.

Complete Example

[code=php] public function submit(): void { // Set validation rules $this->validation->set_rules('first_name', 'first name', 'required|min_length[2]|max_length[50]'); $this->validation->set_rules('email', 'email address', 'required|valid_email'); $this->validation->set_rules('birth_date', 'birth date', 'required|valid_date'); $this->validation->set_rules('password', 'password', 'required|min_length[8]'); $this->validation->set_rules('confirm_password', 'password confirmation', 'required|matches[password]'); $this->validation->set_rules('avatar', 'profile photo', 'is_image|square|max_size[5000]'); // Run validation if ($this->validation->run() === true) { // All validation passed - process form $data = $this->model->get_data_from_post(); $this->db->insert($data, 'users'); set_flashdata('Account created successfully'); redirect('login'); } else { // Validation failed - redisplay form with errors $this->create(); } } [/code]
