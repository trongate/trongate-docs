previous_url()

function previous_url(): string

Description

Returns the URL of the previously visited page, if available. This is typically used for "back" links or redirecting users after form submission.

Return Value

Type Description
string The URL of the previous page, or an empty string if no previous URL is available.

Example Usage

[code=php] $back = previous_url(); echo $back; // e.g., "https://example.com/products"[/code]
