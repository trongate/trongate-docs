redirect()

function redirect(string $target_url): void

Description

Performs an HTTP redirect to the specified URL. If the target URL does not start with http, it is assumed to be an internal route and BASE_URL is automatically prepended.

Parameters

Parameter Type Description Default
$target_url string The URL or internal route to redirect to. N/A

Return Value

Type Description
void This function does not return a value; it terminates execution after sending the redirect header.

Example Usage

[code=php] // Redirect to an external URL redirect('https://example.com'); // Redirect to an internal route (BASE_URL is prepended) redirect('products/view/42');[/code]
