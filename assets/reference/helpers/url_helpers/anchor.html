anchor()

function anchor(string $url, ?string $text = null, array $attributes = []): string

Description

Generates an HTML anchor tag (<a>) with automatic base URL resolution and XSS protection. If the URL is relative (does not start with http://, https://, or //), the BASE_URL constant is prepended to ensure the link points to the correct internal route.

Attributes and URLs are automatically escaped to prevent XSS attacks.

Parameters

Parameter Type Description Default
$url string The destination URL. Can be relative or absolute. N/A
$text ?string Optional. The visible link text. If null, the URL is used as the link text. null
$attributes array Optional. An associative array of HTML attributes to apply to the anchor tag, e.g., ['class' => 'btn']. []

Return Value

Type Description
string The generated HTML anchor tag.

Example Usage

[code=php] // Basic link with auto-resolved URL echo anchor('products/view/42', 'View Product'); // Output: View Product // Link with HTML attributes echo anchor('login', 'Sign In', ['class' => 'btn btn-primary']); // Output: Sign In // External link (no BASE_URL prepended) echo anchor('https://example.com', 'External Site'); // Output: External Site // Using URL as link text echo anchor('contact'); // Output: contact[/code]
