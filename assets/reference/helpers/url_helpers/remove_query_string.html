remove_query_string()

function remove_query_string(string $string): string

Description

Removes the query string from a given URL and returns the clean URL.

Parameters

Parameter Type Description Default
$string string The URL with a query string to be processed. N/A

Return Value

Type Description
string The URL without the query string.

Example Usage

[code=php] $url = "https://example.com/page?name=test&id=5"; $clean = remove_query_string($url); echo $clean; // "https://example.com/page"[/code]
