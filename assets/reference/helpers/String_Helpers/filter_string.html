filter_string()

function filter_string(string $string, array $allowed_tags = []): string

Deprecated: This function is deprecated and will be removed from the Trongate framework on June 3, 2026. Developers are encouraged to globally replace instances of 'filter_string(' with 'filter_str(' throughout their site or application.

Description

Alias for filter_str() for backward compatibility. Filters and sanitizes a string, removing any disallowed HTML tags while preserving allowed ones.

While similar functionality can be achieved using the out() function, filter_string() offers control over which HTML tags are allowed or removed.

Parameters

Parameter Type Description
$string string The input string to be filtered and sanitized.
$allowed_tags array Optional. An array of allowed HTML tags. Default is an empty array.

Return Value

Type Description
string The filtered and sanitized string.

Example Usage

$input_string = '<script>alert("Hello");</script>';
$allowed_tags = ['<p>', '<a>'];
echo filter_string($input_string, $allowed_tags);
// Output: 'alert("Hello");'
