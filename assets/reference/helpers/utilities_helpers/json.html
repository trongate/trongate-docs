json()

function json(mixed $data, ?bool $kill_script = null): void

Description

Outputs the given data as JSON in a prettified format, suitable for debugging and visualization. This function is especially useful during development for inspecting data structures in a readable JSON format directly in the browser.

It optionally allows terminating the script immediately after output, useful in API development for stopping further processing.

Parameters

Parameter Type Description Default
$data mixed The data (array, object, or other JSON-encodable value) to encode and output. N/A
$kill_script ?bool Optional. If true, script execution halts immediately after JSON output. Default is null (script continues). null

Return Value

Type Description
void Output is written directly to the response buffer.

Example Usage

[code=php] // Output data as JSON and continue execution $data = ['name' => 'Widget', 'price' => 19.99]; json($data); // Output data and stop script execution json($data, true);[/code]
