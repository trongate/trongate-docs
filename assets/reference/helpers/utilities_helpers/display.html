display()

function display(array $data): void

Description

Renders a content view within a template. This function expects an associative array containing the target view module and view file, and outputs the rendered content directly.

Parameters

Parameter Type Description Default
$data array An associative array containing view_module (the module name) and view_file (the view file name without extension). N/A

Return Value

Type Description
void Output is written directly to the response buffer.

Example Usage

[code=php] $data = [ 'view_module' => 'products', 'view_file' => 'product_list' ]; display($data); // Renders modules/products/views/product_list.php[/code]
