block_url()

function block_url(string $block_path = ''): void

Description

Blocks direct browser access to a module or a specific module-method combination. This helper prevents certain code from being invoked via URL access while allowing unrestricted internal PHP calls. It compares the current URL segments against the supplied target path and returns a 403 Forbidden response if a match is found.

Parameters

Parameter Type Description Default
$block_path string The module or module/method string to block. Use 'module' to block all methods, or 'module/method' to block a specific method. ''

Return Value

Type Description
void This function does not return a value; it terminates execution after issuing a 403 Forbidden response if the path matches.

Example Usage

[code=php] // Block all methods in the 'payment' module block_url('payment'); // Block only the 'process' method in 'payment' block_url('payment/process');[/code]
