sort_rows_by_property()

function sort_rows_by_property(array $array, string $property, string $direction = 'asc'): array

Description

Sorts an array of objects by a specified property. Unlike sort_by_property(), this function works with objects rather than associative arrays.

Parameters

Parameter Type Description Default
$array array The array of objects to be sorted. N/A
$property string The property by which to sort the objects. N/A
$direction string The sort direction. Use 'asc' for ascending or 'desc' for descending. 'asc'

Return Value

Type Description
array The sorted array of objects.

Example Usage

[code=php] $users = [ (object) ['name' => 'Alice', 'age' => 30], (object) ['name' => 'Bob', 'age' => 25], (object) ['name' => 'Charlie', 'age' => 35] ]; // Sort by age ascending $sorted = sort_rows_by_property($users, 'age'); // Sort by name descending $sorted = sort_rows_by_property($users, 'name', 'desc');[/code]
