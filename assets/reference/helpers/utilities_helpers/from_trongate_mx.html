from_trongate_mx()

function from_trongate_mx(): bool

Description

Checks whether the current HTTP request originated from a Trongate MX (Mobile Experience) client. It returns true if the request headers contain X-Trongate-MX: true, indicating the request was made by a Trongate MX mobile app.

Return Value

Type Description
bool True if the request has the X-Trongate-MX header set to 'true', otherwise false.

Example Usage

[code=php] if (from_trongate_mx()) { // Serve a mobile-optimised response display(['view_module' => 'products', 'view_file' => 'mobile_list']); } else { // Serve standard web response display(['view_module' => 'products', 'view_file' => 'web_list']); }[/code]
