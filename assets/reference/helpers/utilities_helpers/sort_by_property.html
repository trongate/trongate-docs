sort_by_property()

function sort_by_property(array &$array, string $property, string $direction = 'asc'): array

Description

Sorts an array of associative arrays by a specified property. The input array is sorted in place and the sorted array is also returned.

Parameters

Parameter Type Description Default
$array array The array to be sorted (passed by reference). N/A
$property string The property key by which to sort the array. N/A
$direction string The sort direction. Use 'asc' for ascending or 'desc' for descending. 'asc'

Return Value

Type Description
array The sorted array.

Example Usage

[code=php] $products = [ ['name' => 'Widget', 'price' => 19.99], ['name' => 'Gadget', 'price' => 9.99], ['name' => 'Doohickey', 'price' => 29.99] ]; // Sort by price ascending sort_by_property($products, 'price'); // Sort by name descending sort_by_property($products, 'name', 'desc');[/code]
