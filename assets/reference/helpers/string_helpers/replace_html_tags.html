replace_html_tags()

function replace_html_tags(string $content, array $specifications): string

Description

Converts HTML content by replacing specified tag patterns with new tags. The specifications array defines the opening and closing strings to find, and what to replace them with.

Parameters

Parameter Type Description Default
$content string The original HTML content to be converted. N/A
$specifications array An associative array with keys: opening_string_before, close_string_before, opening_string_after, close_string_after. N/A

Return Value

Type Description
string The modified HTML content with tags replaced.

Example Usage

[code=php] $html = '<span class="highlight">Important</span>'; $specs = [ 'opening_string_before' => '<span class="highlight">', 'close_string_before' => '</span>', 'opening_string_after' => '<strong>', 'close_string_after' => '</strong>' ]; echo replace_html_tags($html, $specs); // Outputs: <strong>Important</strong>[/code]
