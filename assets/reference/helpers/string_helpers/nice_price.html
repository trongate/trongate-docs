nice_price()

function nice_price(float $num, ?string $currency_symbol = null): string

Description

Formats a number as a price with commas for thousands and optionally adds a currency symbol. If the formatted price is a whole number, decimals are removed.

Parameters

Parameter Type Description Default
$num float The number to be formatted as a price. N/A
$currency_symbol string|null Optional. The currency symbol to prepend to the price. null

Return Value

Type Description
string Returns the formatted price as a string with thousands separators. Currency symbol added if specified.

Example Usage

[code=php] echo nice_price(123456.00); // Output: 123,456 echo nice_price(123456.78, '$'); // Output: $123,456.78[/code]
