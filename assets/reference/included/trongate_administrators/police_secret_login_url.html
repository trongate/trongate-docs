police_secret_login_url()

public function police_secret_login_url(): string

Description

Returns the login form submission URL, supporting an optional "secret login segment" feature. If a secret login segment is configured, accessing the module URL directly triggers a 404 to obscure the admin login path.

This is a security feature that prevents automated bots from finding the admin login page by using an obfuscated URL.

Parameters

This method accepts no parameters (uses $this->secret_login_segment if set).

Return Value

TypeDescription
stringThe login form submission URL (e.g., trongate_administrators/submit_login or a custom secret segment).

Example Usage

[code=php] // Default usage (no secret segment) $login_url = $this->trongate_administrators->police_secret_login_url(); // Returns: "trongate_administrators/submit_login" // With secret segment configured in controller: public $secret_login_segment = 'my-secret-admin-login'; // Then police_secret_login_url() returns: "my-secret-admin-login/submit_login" // And accessing "trongate_administrators/login" triggers a 404 // Used internally by login() method: public function login(): void { $data['form_location'] = $this->police_secret_login_url(); $this->trongate_tokens->destroy(); $this->view('login', $data); } [/code]
