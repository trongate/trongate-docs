delete_conf()

public function delete_conf(): void

Description

Displays a confirmation page before deleting a record. Prevents users from deleting their own account. Requires a valid update ID in URL segment 3.

Parameters

This method accepts no parameters (uses URL segment 3 for the record ID).

Return Value

TypeDescription
voidOutput rendered via the admin template, or redirects on self‑deletion attempt.

Example Usage

[code=php] // URL: /trongate_administrators/delete_conf/42 // Shows confirmation for deleting record #42 // In a view file (delete confirmation link) <a href="<?= BASE_URL ?>trongate_administrators/delete_conf/<?= $update_id ?>" class="btn btn-danger">Delete Record</a> // From another controller public function prompt_delete(int $id) { redirect('trongate_administrators/delete_conf/' . $id); } [/code]
