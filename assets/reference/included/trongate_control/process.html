process()

public function process(): void

Description

Handles AJAX POST requests for SQL file operations. Accepts action data from the management interface and routes it to the appropriate handler. Supports actions such as viewing SQL content, executing SQL files, and deleting SQL files.

Only operates in development mode. Returns appropriate HTTP status codes for error conditions (400 for bad requests, 404 for missing files).

Parameters

This method takes no parameters. The action data is retrieved from the POST request body.

Return Value

TypeDescription
voidOutputs the result of the SQL operation. Returns a 400 status code for bad requests.
