write()

function write(string $file_path, mixed $data, bool $append = false): bool

Description

Writes or appends data to a file.

Parameters

Parameter Type Description Default
$file_path string The path to the file where data should be written. N/A
$data mixed The data to write to the file. N/A
$append bool Optional. Whether to append data to the file instead of overwriting it. false

Return Value

Type Description
bool Returns true on successful write, false on failure.

Throws

Exception if there is an error writing to the file or if access to the path is restricted.

Example Usage

$file_path = '/path/to/your/file.txt';
$data = 'Hello, world!';
try {
    $result = write($file_path, $data);
    if ($result) {
        echo 'Data written successfully!';
    } else {
        echo 'Failed to write data to the file.';
    }
} catch (Exception $e) {
    echo 'Error: ' . $e->getMessage();
}
