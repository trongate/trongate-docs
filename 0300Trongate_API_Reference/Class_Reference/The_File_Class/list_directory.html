list_directory()

public function list_directory(string $directory_path, bool $recursive = false): array

Description

Lists files and directories within a specified directory.

This method returns an array containing the names of files and directories within the specified directory. It has an option to perform a recursive listing, which includes all subdirectories and their contents.

Parameters

Parameter Type Description Default
$directory_path string The path to the directory whose contents are to be listed. N/A
$recursive bool Optional. Determines whether the listing should be recursive. false

Throws

Type Description
Exception If the specified directory does not exist or is not a directory, or if access to the directory is restricted.

Return Value

Type Description
array An array of file and directory names from the specified directory.

Example Usage

$obj = new YourClass();
$directory_path = "/path/to/directory";
$recursive = true;
try {
    $result = $obj->list_directory($directory_path, $recursive);
    echo "Contents of directory $directory_path:
";
    print_r($result);
} catch (Exception $e) {
    echo "Error: " . $e->getMessage();
}
