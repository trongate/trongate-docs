read()

public function read(string $file_path): string

Description

Reads the contents of a file.

Parameters

Parameter Type Description Default
$file_path string The path to the file to be read. N/A

Return Value

Type Description
string Returns the contents of the file.

Throws

Type Description
Exception If the file does not exist or cannot be read, or if access to the file is restricted.

Example Usage

$obj = new YourClass();
$file_path = "/path/to/file.txt";
try {
    $content = $obj->read($file_path);
    echo "File content: $content";
} catch (Exception $e) {
    echo "Error: " . $e->getMessage();
}
