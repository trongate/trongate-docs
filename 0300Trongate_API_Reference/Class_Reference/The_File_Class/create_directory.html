create_directory()

public function create_directory(string $directory_path, int $permissions = 0755, bool $recursive = true): bool

Description

Creates a new directory at the specified path, allowing for the creation of nested directories if they do not exist.

Parameters

Parameter Type Description Default
$directory_path string The path where the directory should be created. N/A
$permissions int Optional. The permissions to set for the directory, in octal notation (e.g., 0755). 0755
$recursive bool Optional. Whether to create nested directories if necessary. true

Return Value

Type Description
bool Returns true if the directory was created successfully, or if it already exists.

Throws

Type Description
Exception If the directory cannot be created, or if access to the path is restricted.

Example Usage

$obj = new YourClass();
$result = $obj->create_directory("/path/to/new/directory", 0755, true);
if ($result) {
    echo "Directory created successfully!";
} else {
    echo "Failed to create directory.";
}
