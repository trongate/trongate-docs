info()

public function info(string $file_path): array

Description

Retrieves metadata about a file, including its size, last modification time, permissions, file name, and MIME type.

This method provides information about a file including its size, last modification time, and permissions. Additionally, it now returns the file name and MIME type.

Parameters

Parameter Type Description Default
$file_path string The path to the file. N/A

Return Value

Type Description
array Returns an array with file metadata.

Throws

Type Description
Exception If the file does not exist.

Example Usage

$info = $obj->info("/path/to/file.txt");
print_r($info);
/* Output:
Array
(
    [file_name] => file.txt
    [size] => 1024
    [modified_time] => 1620379200
    [permissions] => 33206
    [mime_type] => text/plain
    [readable_permissions] => 0646
    [human_readable_size] => 1.00 KB
)
*/
