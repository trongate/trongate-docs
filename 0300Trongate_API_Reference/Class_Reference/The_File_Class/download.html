download()

public function download(string $file_path, bool $as_attachment = true): void

Description

Initiates a file download or displays inline from the server or an external URL.

This method prepares and sends headers based on the parameters to either initiate a file download from the server's local storage or display it inline. It checks if the file exists and is readable before proceeding.

Parameters

Parameter Type Description Default
$file_path string The path or URL of the file. N/A
$as_attachment bool Optional. Determines whether to force the file download (true) or display inline (false). true

Throws

Type Description
Exception If the file does not exist, cannot be read, or if access to the file is restricted.
