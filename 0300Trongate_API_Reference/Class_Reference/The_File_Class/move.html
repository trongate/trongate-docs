move()

public function move(string $source_path, string $destination_path): bool

Description

Moves a file from one location to another.

Parameters

Parameter Type Description Default
$source_path string The path to the source file. N/A
$destination_path string The path to the destination where the file will be moved. N/A

Throws

Type Description
Exception If the source file does not exist or if the move operation fails.

Return Value

Type Description
bool Returns true on success, or false on failure.

Example Usage

try {
    $obj = new YourClass();
    $source_path = "/path/to/source/file";
    $destination_path = "/path/to/destination/file";
    $result = $obj->move($source_path, $destination_path);
    if ($result) {
        echo "File moved successfully.";
    } else {
        echo "Failed to move the file.";
    }
} catch (Exception $e) {
    echo "Error: " . $e->getMessage();
}
