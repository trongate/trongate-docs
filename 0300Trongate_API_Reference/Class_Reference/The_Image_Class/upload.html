upload()

public function upload(array $data): array

Description

Handles the uploading of files, including image processing and thumbnail creation.

Parameters

Parameter Type Description
$data array An associative array containing upload parameters such as destination, maximum width, maximum height, thumbnail directory, and more.

Return Value

An associative array containing information about the uploaded file, including its name, path, type, size, and optionally, the thumbnail path.

Exceptions

This method may throw an exception if any errors occur during the upload process, such as invalid directory, file already exists, or other file-related issues.

Example Usage

$data = [
    'destination' => 'uploads',
    'max_width' => 800,
    'max_height' => 600,
    'thumbnail_dir' => 'thumbnails',
    'thumbnail_max_width' => 200,
    'thumbnail_max_height' => 150,
    'upload_to_module' => true,
    'make_rand_name' => true,
];
try {
    $file_info = upload($data);
    echo "File uploaded successfully: " . $file_info['file_name'];
} catch (Exception $e) {
    echo "An error occurred: " . $e->getMessage();
}
