upload_picture()

public function upload_picture(array $data): ?array

Description

Uploads a picture file.

Parameters

Parameter Type Description Default
$data array The data for the uploaded file. N/A

Return Value

Type Description
array|null The information of the uploaded file, or null if the upload fails.

Example Usage

$result = upload_picture($data);
// $result contains the information of the uploaded picture file, or null if the upload fails.
