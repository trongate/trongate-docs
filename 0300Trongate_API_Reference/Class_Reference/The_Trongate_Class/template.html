template()

public function template(string $template_name, array $data = []): void

Description

Loads a template controller file, instantiates the corresponding object, and calls the specified template method with the given data.

Parameters

Parameter Type Description Default
$template_name string The name of the template method to call. N/A
$data array The data to pass to the template method. []

Throws

Type Description
Exception If the template controller file cannot be found or the template method does not exist.

Return Value

This method does not return a value.
