form_datetime()

function form_datetime(string $name, string|null $value = null, ?array $attributes = null, ?string $additional_code = null): string

Description

Generates an HTML input element with type "datetime-local".

Parameters

Parameter Type Description
$name string The name attribute for the input element.
$value string|null The initial value of the input element (datetime-local). Use null for no initial value.
$attributes array|null An associative array of HTML attributes for the input.
$additional_code string|null Additional HTML code to append to the input element.

Return Value

Type Description
string The generated HTML input element with type "datetime-local".

Example Usage

$name = 'appointment';
$value = '2023-10-20T14:30';
$attributes = ['min' => '2022-01-01T00:00', 'max' => '2024-12-31T23:59'];
$additional_code = 'required';
echo form_datetime($name, $value, $attributes, $additional_code);
// Output: '<input type="datetime-local" name="appointment" value="2023-10-20T14:30" min="2022-01-01T00:00" max="2024-12-31T23:59" required>'
